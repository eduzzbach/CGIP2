Keys:
'h' - Toggle this panel
'0' - Toggle multiple views
'1' - Front view
'2' - Left view
'3' - Top view
'4' - Axonometric view
'8' - Toggle Axonometric/Oblique View
'9' - Toggle Parallel/Perspective View
' ' - Toggle wireframe/solid
'q' - Move forward
'e' - Move backwards
'w' - Raise Cannon
's' - Lower Cannon
'a' - rotate cabin counterclockwise
'd' - rotate cabin clockwise
'r' - Reset view params
'ArrowLeft' - Increase theta
'ArrowRight' - Decrease theta
'ArrowUp' - Increase gamma
'ArrowDown' - Decrease gamma
